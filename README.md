# ⚡ NEXWRA — Autonomous AI Workforce Platform

> The future of work isn't one AI. It's a team of AI agents working together.

NEXWRA is a next-generation AI orchestration platform that enables multiple autonomous AI agents to collaborate, debate, research, analyze, and execute complex tasks in parallel.

Instead of relying on a single chatbot, NEXWRA creates specialized AI teams where each agent has its own role, memory, and objective — working together like a real digital workforce.

## 🚀 Core Vision

Imagine hiring an entire team of experts with a single prompt.

NEXWRA transforms one task into a coordinated mission:

* 🔎 Research Agents gather information
* 🧠 Strategy Agents analyze possibilities
* ✍️ Writer Agents generate polished outputs
* ⚡ Executor Agents perform actions
* 🧩 Coordinator Agents orchestrate workflows

The result is faster, smarter, and more reliable problem solving.

# ⚙️ Local Development

## Clone Repository

```bash
git clone https://github.com/vedx-bit/NEXWRA.git
cd NEXWRA
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```

## Backend

```bash
pip install -r requirements.txt
streamlit run app.py
```
